1. Introduction
This project presents the design of a mobile application prototype for a food delivery service named Fatfood. The goal of the project is to create a visually appealing and user-friendly interface using modern UI/UX design principles.
The prototype was developed using Figma and focuses on a simple but meaningful user flow, allowing users to browse food and view product details.

2. Design Concept
The application is designed to provide a smooth and intuitive experience for users who want to order food quickly. The interface emphasizes simplicity, clarity, and accessibility.
The main concept includes:
+ Clean layout with minimal clutter
+ Strong visual hierarchy
+ Easy navigation between screens
+ Consistent design elements

3. Style Guide
A style guide was created to ensure consistency across all screens.

3.1 Color Palette
The following color scheme was applied:
Primary Color: #FF6B35 (used for main actions such as buttons)
Secondary Color: #2EC4B6 (used for tags and secondary elements)
Background Color: #FFFFFF (used for the main background)
Text Color: #1E1E1E (used for readability)
Accent Color: #FFE66D (used for highlighting important information such as price and rating)
This color palette helps create a modern and visually balanced interface.

3.2 Typography
The application uses the Poppins font for consistency and readability.
Heading 1: 24px Bold
Heading 2: 18px SemiBold
Body Text: 14px Regular
Button Text: 16px Medium
The typography system ensures clear hierarchy and improves user experience.

3.3 UI Components
Reusable components were created to maintain consistency:
Buttons (Primary action button)
Input fields (for login screen)
Food cards (for displaying products)
Using components improves efficiency and ensures design consistency.

5. Prototype Flow

A simple interaction flow was created:

Welcome → Sign In
Sign In → Home
Home → Food Detail

This flow simulates a basic user journey from entering the app to viewing a product.

6. Design Principles Applied

The design follows key UI/UX principles:

Consistency: Same colors, fonts, and components across all screens
Visual Hierarchy: Important elements such as buttons and titles are emphasized
Simplicity: Minimal and clean layout
Usability: Easy navigation and clear actions

7. Conclusion

This project demonstrates the process of designing a simple mobile app prototype using Figma. By applying UI/UX principles and a structured design system, the final product achieves both visual quality and usability.

The design can be further improved by adding more features such as a cart system, payment process, and user profile.
