Homepage (Landing Page)
Purpose: This is the main introductory page of the WE LOVE CARDS website.
Explanation:
The top navigation bar includes:
-Homepage: return to the main page
-Marketplace: leads to the card marketplace
-Sell: allows users to sell cards
-Login: user authentication
A large banner in the middle displays attractive card images to grab the user's attention.
The main title “ONLINE Card Trading” clearly explains the website's function.
Supporting text highlights key features:
-Buy rare cards
-Sell your collection
-Secure transactions
The “Go to Marketplace” button is a call to action guiding users to start browsing cards.

Market Page
Purpose:This page is where users browse, buy, and trade cards.
Explanation:
The header displays “Exchange – Secure and Safe Trading,” emphasizing trust and security.
The cards are displayed in a grid layout, making it easy to quickly browse the information.
Each card entry includes:
-Card Image
-Card Name (MLBB Card)
-Price (VND)
Action Buttons:
-Buy – purchase cards
-Trade – trade cards with other users
-Different price levels represent different card values, from common to rare.
This page focuses on e-commerce functionality and user interaction.

Sell Cards Page
Purpose: This page allows users to list their own cards for sale.
Explanation: The title “Sell Cards – Earn from your collection” motivates users to sell.
The form includes:
-Card Name – identify the card
-Price (VND/card) – selling price
-Quantity – number of cards
-Total Price (VND) – calculated total cost
-Card Image Upload – helps buyers see the product
-Sell/Trade option – choose selling method
The Submit button uploads the listing to the marketplace.
This page represents the seller side of the platform.

Login Page
Purpose:This page is used for user authentication and account access.

Centered login card improves focus and usability.

-Forgot password? – account recovery

This page focuses on security, access control, and user management.
