1. Introduction
In this lab, the Mobirise website builder was used to create a simple website interface based on Bootstrap templates. The purpose of the website is to present a collection of Blokees toys using a clean and responsive layout.
2. Tools and Technologies
The following tools and technologies were used in this project:
Bootstrap Framework – used for responsive layout and UI components
Mobirise Website Builder – used to create the website using drag-and-drop blocks
HTML / CSS / JavaScript – generated automatically by Mobirise
Images of Blokees toys – used for visual presentation of the products
Mobirise provides many pre-designed Bootstrap blocks such as menus, headers, cards, galleries, and forms. These blocks help simplify the development process.
4. Implementation
3.1 Website Structure
The website was designed with the following layout:
Navigation Bar
Hero Section (Header)
Toy Categories
Product Cards
Image Gallery
Contact Form
Footer
This structure creates a clear and user-friendly interface.
3.2 Navigation Bar
A navigation menu was added at the top of the page to help users move between different sections of the website.
Menu items include:
Home
Toys
Collection
Gallery
Contact
This component was created using a Bootstrap menu block provided by Mobirise.
3.3 Hero Section
The hero section introduces the website and attracts users' attention. It contains:
